# Old World RPG

## Project Documentation

### Introduction
Old World RPG is a text-based role-playing game set in a mythical world full of ancient lore, adventures, and quests. Players can create their characters, embark on quests, and interact with various NPCs in a richly developed environment.

### Features
- Character creation customization
- Engaging storyline with multiple quests
- Interaction with diverse NPCs
- Combat system for battling foes
- Inventory management for items and equipment

### Installation
To install Old World RPG, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/akingdom708-hub/old-world-rpg.git
   ```
2. Navigate to the project directory:
   ```bash
   cd old-world-rpg
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the game:
   ```bash
   npm start
   ```

### Usage
1. Launch the game using the terminal.
2. Follow the on-screen instructions to create your character.
3. Explore the world, engage in quests, and interact with other characters.
